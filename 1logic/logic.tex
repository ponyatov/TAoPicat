\secrel{Logic Programs\\Логические программы}\secdown

A logic program is a set of axioms, or rules, defining relations between objects. 
Computation of a logic program is a deduction of consequences of the program. 
A program defines a set of consequences, which is its meaning. 
The art of logic programming is construction concise and elegant programs that have the desired meaning.

\ru{
Логическая программа\ --- это набор аксиом или правил, определяющих отношения между объектами. 
Вычисление логической программы является выводом следствий из этой программы. 
Программа определяет набор следствий, которые является ей смыслом. 
Искусство логического программирования заключается в построении лаконичных и элегантных программ, которые имеют желаемый смысл.
}
